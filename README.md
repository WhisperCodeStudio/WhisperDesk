# WhisperDesk

WhisperDesk is a lightweight productivity dashboard designed to organise tasks, projects, and workflow in a calm, focused interface.

## Features

• Task management  
• Project organisation  
• Dashboard overview  
• Supabase backend integration

## Tech Stack

• Next.js  
• TypeScript  
• Supabase  
• TailwindCSS

## Development

Run locally:
npm install
npm run dev

Then open:

http://localhost:3000

## Part of the WhisperCode ecosystem

WhisperDesk is developed as part of the WhisperCode studio alongside projects such as SymptoSync and WhisperVerse.
